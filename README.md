# Tetris

## Build & Run

### Clone repository
```bash
git clone https://github.com/Navsika/tetris.git  
cd tetris
```

### Build with Maven
```bash
mvn clean package
```

### Run

**Graphical mode (default):**  
```bash
java -jar target/tetris-1.0.jar
```

**Console mode:**  
```bash
java -jar target/tetris-1.0.jar --console
```
